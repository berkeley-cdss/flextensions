## Specific Insructor Setup Documentation Potential Edits
### COURSE EXTENSION REQUESTS SECTION:
- **IMPORTANT:** The documentation prompts instructors to "Click View next to any request to access its full details." However, there is no "View" button next to any of the requests on the requests page. Instead, this should prompt instructors to click the student's name associated with the request that they would like to view the details of.
  - The same goes for the instructions for editing extension requests, as well as the instructions for Method 2 of accepting or denying requests
-   **Anotheer potential fix could be to actually add a View button for each request**

### VIEWING REQUEST HISTORY SECTION:
